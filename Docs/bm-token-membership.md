## 1. BM 개요

앱의 프리미엄 기능에 **토큰 소모** 또는 **멤버십 등급**에 따른 접근 권한을 부여하여 수익을 창출합니다.

### 수익 모델 구조

```
┌──────────────────────────────────────────────────────────────┐
│                        수익 모델                              │
├──────────────────────────┬───────────────────────────────────┤
│     토큰 (Token)          │       멤버십 (Membership)          │
│                          │                                   │
│  • 건당 과금 (Pay-per-use) │  • 월정액 구독 (Subscription)      │
│  • 소량 구매 가능          │  • 등급별 혜택 차등                │
│  • 진입 장벽 낮음          │  • 장기 고정 수익                  │
│  • 캐주얼 사용자 타겟      │  • 헤비 사용자 타겟               │
└──────────────────────────┴───────────────────────────────────┘
```

### 프리미엄 기능 요약

| #   | 기능                 | 설명                                    | 과금 방식     |
| --- | -------------------- | --------------------------------------- | ------------- |
| 1   | 포트폴리오 백테스팅  | 과거 데이터 기반 수익률 시뮬레이션      | 토큰 / 멤버십 |
| 2   | 포트폴리오 카피      | 커뮤니티 공유 포트폴리오 가져오기       | 토큰 / 멤버십 |
| 3   | 포트폴리오 상세 열람 | AI 평가 점수, Sharpe Ratio 등 상세 지표 | 토큰 / 멤버십 |
| 4   | AI 희망 비중 추천    | AI 기반 종목 비중 최적화 추천           | 토큰 / 멤버십 |

## 2. 기능별 상세 설계

### 2-1. 📊 포트폴리오 백테스팅

#### 기능 설명

과거 시장 데이터를 기반으로 "내 포트폴리오가 과거에 어떤 성과를 냈을지" 시뮬레이션하는 기능입니다.

#### 상세 기능 구성

| 항목                   | 설명                                                             |
| ---------------------- | ---------------------------------------------------------------- |
| **백테스팅 기간 설정** | 시작일~종료일 지정 (최소 1개월 ~ 최대 10년)                      |
| **대상 설정**          | 개별 종목 또는 전체 포트폴리오 단위 선택                         |
| **리밸런싱 주기 설정** | 월 1회, 분기 1회, 반기 1회, 연 1회 등                            |
| **리밸런싱 조건 설정** | 희망 비중 대비 현재 비중 N% 이상 차이 시 리밸런싱 발동 (예: 20%) |
| **수익률 그래프**      | 기간별 누적 수익률, 벤치마크(코스피/S&P500) 대비 비교 차트       |
| **비교 분석**          | 여러 백테스팅 결과를 겹쳐서 비교 (조건 변경에 따른 차이 확인)    |

#### 결과 제공 지표

| 지표                 | 설명                            | 무료 공개 | 프리미엄 전용 |
| -------------------- | ------------------------------- | :-------: | :-----------: |
| 누적 수익률          | 전체 기간 총 수익률             |    ✅     |       -       |
| 연평균 수익률 (CAGR) | 연 단위 환산 수익률             |    ✅     |       -       |
| 최대 낙폭 (MDD)      | 최고점 대비 최대 하락률         |     -     |      ✅       |
| 변동성 (Volatility)  | 수익률의 표준편차               |     -     |      ✅       |
| Sharpe Ratio         | 위험 대비 수익률 비율           |     -     |      ✅       |
| 리밸런싱 이력        | 리밸런싱 발동 시점 및 매매 내역 |     -     |      ✅       |
| 벤치마크 비교        | 코스피/S&P500 대비 초과 수익    |     -     |      ✅       |

#### 사용자 플로우

```
포트폴리오 선택
    │
    ▼
"백테스팅" 버튼 클릭
    │
    ▼
┌─────────────────────────┐
│  백테스팅 설정           │
│  • 기간: 2021.01~2025.12│
│  • 리밸런싱: 분기 1회    │
│  • 조건: 비중 20% 차이   │
└────────┬────────────────┘
         │
         ▼
    토큰 차감 / 멤버십 확인
         │
         ▼
┌─────────────────────────┐
│  결과 화면               │
│  📈 수익률 그래프        │
│  📊 상세 지표            │
│  🔄 비교 분석 추가       │
└─────────────────────────┘
```

#### BM 적용 방식

| 항목                  | 토큰   | 멤버십                                       |
| --------------------- | ------ | -------------------------------------------- |
| 1회 백테스팅 실행     | 5 토큰 | Basic: 3회/월, Pro: 15회/월, Premium: 무제한 |
| 비교 분석 (추가 건당) | 2 토큰 | Basic: 1회/월, Pro: 5회/월, Premium: 무제한  |
| 상세 지표 열람        | 포함   | 포함                                         |

### 2-2. 📋 포트폴리오 카피

#### 기능 설명

커뮤니티에서 다른 사용자가 공유한 포트폴리오 구성(종목 및 비중)을 내 포트폴리오로 가져오는 기능입니다.

#### 상세 기능 구성

| 항목                     | 설명                                                                        |
| ------------------------ | --------------------------------------------------------------------------- |
| **공유 포트폴리오 탐색** | 커뮤니티 피드에서 공유된 포트폴리오 목록 확인 (무료)                        |
| **기본 정보 미리보기**   | 포트폴리오명, 전체 수익률, 종목 수, 생성자 (무료)                           |
| **카피 실행**            | 선택한 포트폴리오의 종목 구성과 희망 비중을 내 포트폴리오로 복사 (프리미엄) |
| **커스터마이징**         | 카피 후 개별 종목 비중 수정 가능                                            |
| **카피 출처 표시**       | 카피한 포트폴리오에 원본 출처 링크 표시                                     |

#### 무료 vs 프리미엄 차이

| 요소                             | 무료 사용자 | 프리미엄 사용자 |
| -------------------------------- | :---------: | :-------------: |
| 공유 포트폴리오 목록 조회        |     ✅      |       ✅        |
| 기본 정보 확인 (수익률, 종목 수) |     ✅      |       ✅        |
| 포트폴리오 카피 실행             |     ❌      |       ✅        |
| 카피 후 비중 수정                |      -      |       ✅        |

#### BM 적용 방식

| 항목                | 토큰   | 멤버십                                      |
| ------------------- | ------ | ------------------------------------------- |
| 포트폴리오 1건 카피 | 3 토큰 | Basic: 1회/월, Pro: 5회/월, Premium: 무제한 |

### 2-3. 🔍 포트폴리오 상세 내용 열람

#### 기능 설명

커뮤니티에서 공유된 포트폴리오의 **심화 분석 지표**를 열람하는 기능입니다.
기본 공개 정보 외에 AI 포트폴리오 평가, 리스크 지표 등을 확인할 수 있습니다.

#### 공개 범위 설계

| 정보 레벨     |  공개 범위   | 포함 항목                                                |
| ------------- | :----------: | -------------------------------------------------------- |
| **Lv.1 기본** | 🌐 전체 공개 | 포트폴리오명, 전체 수익률, 희망 비중 차트, 종목 수       |
| **Lv.2 상세** | 🔒 프리미엄  | AI 포트폴리오 평가 점수, Sharpe Ratio, 변동성, 상관관계  |
| **Lv.3 심층** | 🔒 프리미엄  | 섹터/국가 분산도, MDD, 리밸런싱 히스토리, AI 리스크 진단 |

#### Lv.2~3 상세 열람 시 제공 지표

| 카테고리        | 지표                 | 설명                            |
| --------------- | -------------------- | ------------------------------- |
| **AI 평가**     | AI 포트폴리오 점수   | A~F 등급 + 100점 만점 점수      |
|                 | AI 리스크 진단       | 집중 리스크, 섹터 편중 등 경고  |
|                 | AI 개선 제안         | "OO 종목 비중 조정 권장" 등     |
| **수익 지표**   | Sharpe Ratio         | 위험 대비 수익 효율성           |
|                 | Sortino Ratio        | 하방 리스크 대비 수익           |
|                 | 연환산 수익률 (CAGR) | 연 단위 환산 수익률             |
| **리스크 지표** | 변동성 (Volatility)  | 수익률 표준편차                 |
|                 | 최대 낙폭 (MDD)      | 최고-최저 구간 하락률           |
|                 | 베타 (Beta)          | 시장 대비 변동 민감도           |
| **분산 지표**   | 섹터 분산도          | IT/금융/헬스케어 등 섹터별 비중 |
|                 | 종목 간 상관관계     | 종목들 사이의 수익률 상관도     |
|                 | 집중도 (HHI)         | 상위 종목 집중도 지수           |

#### BM 적용 방식

| 항목          | 토큰   | 멤버십                                       |
| ------------- | ------ | -------------------------------------------- |
| 1건 상세 열람 | 2 토큰 | Basic: 5건/월, Pro: 20건/월, Premium: 무제한 |

> [!TIP]
> 멤버십 등급이 올라갈수록 **월간 열람 가능 건수**가 증가하는 구조로, 헤비 사용자의 프리미엄 전환을 유도합니다.

### 2-4. 🤖 AI 희망 비중 추천

#### 기능 설명

사용자의 포트폴리오에 포함된 **기존 종목들의 최적 비중**을 AI가 추천합니다.
종목 자체를 추천하는 것이 아니라, 이미 담긴 종목들의 **비중 배분만 최적화**합니다.

> [!IMPORTANT]
> **종목 추천은 하지 않습니다.** 금융 규제 리스크를 피하기 위해, 이미 사용자가 선택한 종목에 대한 비중 조정만 제안합니다.

#### 상세 기능 구성

| 항목                 | 설명                                               |
| -------------------- | -------------------------------------------------- |
| **추천 기반 데이터** | 과거 수익률, 변동성, 종목 간 상관관계, 시장 환경   |
| **최적화 모델**      | Modern Portfolio Theory (MPT) 기반 + AI 보정       |
| **추천 유형**        | 수익 극대화, 리스크 최소화, 균형형 3가지 옵션 제공 |
| **비교 제공**        | 현재 비중 vs AI 추천 비중 시각 비교                |
| **시뮬레이션**       | 추천 비중 적용 시 예상 수익률/리스크 변화 미리보기 |
| **적용 기능**        | "이 비중으로 적용" 버튼으로 즉시 희망 비중 변경    |
