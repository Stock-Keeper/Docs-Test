# PRD Inbox

> 이 디렉토리에 PRD 관련 아이디어/요청을 **자유롭게** 올려주세요.
> 형식, 용어, 문체 신경쓰지 않아도 됩니다!

## 2단계 파이프라인

```
여기 (_inbox/)  →  /prd-prepare  →  _staging/  →  /prd-process  →  specs/
   자유형식          AI 정리        검토/수정       최종 반영       완성
```

## 사용법

### 1단계: 파일 올리기
아무렇게나 작성해서 올리세요:
- "로그인 버튼 누르면 구글로 넘어가게 해줘"
- "결제 기능 추가해야 함"
- "이 화면 좀 이상해" + 스크린샷
- 대화체, 메모, 불완전한 문장 전부 OK

### 2단계: `/prd-prepare` 실행
AI가 분석하여 `_staging/`에 정형화된 초안 생성

### 3단계: `_staging/` 검토
생성된 초안을 확인하고 필요시 수정

### 4단계: `/prd-process` 실행
최종 확인 후 `specs/`에 반영

## 지원 형식

- `.md` - 마크다운
- `.txt` - 텍스트
- `.dbml` - DB 스키마
- `.png`, `.jpg` - 이미지 (설명 추가 권장)
- 기타 텍스트 파일

## 예시 입력

```
# 그냥 이렇게 써도 됨

결제 기능 만들어야 함
- 카드랑 카카오페이
- 월정액 구독 형태로
- 가격은 아직 미정

나중에 환불도 되게 해야 할듯?
```

→ AI가 알아서 API 스펙 초안으로 변환해줌
