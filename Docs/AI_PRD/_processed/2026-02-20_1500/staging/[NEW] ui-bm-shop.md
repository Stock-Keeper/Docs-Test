# 토큰/멤버십 상점

## 개요

사용자가 토큰을 구매하거나 멤버십을 구독하는 화면.

## 레이아웃

```
[ 상단: 나의 상태 ]
보유 토큰: 150 🪙
현재 멤버십: BASIC (D-15) [업그레이드]

[ 탭: 토큰 충전 | 멤버십 구독 ]

(토큰 탭 선택 시)
--------------------------------
[ 소량 패키지 ]     | ₩1,900
 🪙 10개           | [구매]
--------------------------------
[ 인기 패키지 ⭐ ]   | ₩8,900
 🪙 60개 (22%↓)    | [구매]
--------------------------------

(멤버십 탭 선택 시)
--------------------------------
[ Basic ]          | ₩4,900/월
 백테스팅 3회       | [구독]
 카피 1회           |
--------------------------------
[ Pro ]            | ₩9,900/월
 백테스팅 15회      | [구독]
 광고 제거          |
--------------------------------
```

## 컴포넌트

- **StatusCard**: 내 토큰/멤버십 현황 표시
- **ProductList**: 상품 리스트 (탭 전환)
- **PaymentModal**: PG사 결제 창 연동 (WebView or SDK)

## 상호작용

| 액션           | 동작                                                |
| -------------- | --------------------------------------------------- |
| 구매 버튼 클릭 | 결제 모달 오픈 -> PG 결제 진행 -> 완료 시 성공 화면 |
| 탭 전환        | 토큰/멤버십 상품 목록 교체                          |

## 상태

| 상태 | 화면 표시                           |
| ---- | ----------------------------------- |
| 로딩 | 스켈레톤 UI                         |
| 성공 | "구매가 완료되었습니다" + 잔액 갱신 |
| 실패 | "결제에 실패했습니다" + 에러 사유   |
