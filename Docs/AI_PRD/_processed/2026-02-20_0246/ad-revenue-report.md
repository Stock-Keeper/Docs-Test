# 📊 인앱 광고 수익 분석 보고서

> 📅 작성일: 2026-02-14
> 🎯 목적: 앱 내 광고 배치별 사용자 1명당 월 예상 수익 산출

---

## 1. 개요

주식 포트폴리오 관리 + 커뮤니티 앱에 4가지 유형의 광고를 배치하여 발생하는 **사용자 1명당 월간 광고 수익(ARPU)**을 산출합니다.

### 광고 배치 구성

| #   | 위치                 | 광고 유형                  | 비고             |
| --- | -------------------- | -------------------------- | ---------------- |
| 1   | 앱 부팅 시           | 전면 이미지 (Interstitial) | 스플래시 후 노출 |
| 2   | 앱 내 하단           | 텍스트 배너 (Banner)       | 상시 노출        |
| 3   | 커뮤니티 게시물 사이 | 네이티브 인피드 (Native)   | 게시물 5개당 1개 |
| 4   | 알림 페이지 상단     | 고정 배너 (Sticky Banner)  | 고정 위치        |

---

## 2. 분석 전제 조건

### 사용자 행동 가정

| 항목                   | 가정값       | 근거                      |
| ---------------------- | ------------ | ------------------------- |
| 일 평균 앱 실행 횟수   | 2~3회        | 주식 앱 특성 (장 중 확인) |
| 일 평균 체류 시간      | 8~12분       | 투자 앱 평균              |
| 커뮤니티 게시물 스크롤 | 10~15개/세션 | 커뮤니티 활성 사용자 기준 |
| 알림 페이지 방문       | 1회/일       | 일반적 행동 패턴          |

### 시장 기준

| 항목                   | 값                |
| ---------------------- | ----------------- |
| 광고 네트워크          | Google AdMob 기준 |
| 시장                   | 한국 (KR)         |
| 앱 카테고리            | Finance (금융)    |
| 한국 금융 앱 평균 eCPM | $1.0 ~ $5.0       |
| 환율                   | ₩1,400/$          |

> [!NOTE]
> **eCPM** (Effective Cost Per Mille)은 광고 1,000회 노출당 수익입니다.
> 금융 카테고리는 일반 앱 대비 eCPM이 **2~3배** 높은 프리미엄 카테고리에 해당합니다.

| 전면 광고 빈도 제한  | 하루 최대 2회, 콜드 스타트 시 우선 노출 | 이탈률 방지   |
