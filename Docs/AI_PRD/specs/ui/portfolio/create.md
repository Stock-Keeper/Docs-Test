# 포트폴리오 생성 모달 (Portfolio Create)

## 개요

새 포트폴리오 생성 화면 (모달/바텀시트)

## 레이아웃

```text
┌───────────────────────────────────────┐
│  [핸들 -]                             │
│  새 포트폴리오 만들기               [X]│  ← 바텀시트 헤더 (`bottom-sheet-header`)
├───────────────────────────────────────┤
│                                       │
│  이름                                  │
│  ┌─────────────────────────────────┐  │
│  │ 예: 배당주 포트폴리오             │  │
│  └─────────────────────────────────┘  │
│                              0/20자   │
│                                       │
│  설명 (선택)                           │
│  ┌─────────────────────────────────┐  │
│  │ 포트폴리오에 대한 설명            │  │
│  └─────────────────────────────────┘  │
│                             0/100자   │
│                                       │
│  [     취소     ] [    만들기    ]       │  ← 푸터 (`bottom-sheet-footer`)
└───────────────────────────────────────┘
```

## 컴포넌트

### 1. 바텀시트 헤더 (`.bottom-sheet-header`)
- **바텀시트 핸들**: 상단 중앙에 드래그용 핸들 표시 (`.bottom-sheet-handle`)
- **타이틀**: "새 포트폴리오 만들기"
- **닫기 버튼**: `[X]` (`.close-btn`, `#portfolio-modal-close-btn`)

### 2. 바텀시트 바디 (`.bottom-sheet-body`) > 폼 (`.form-group`)

#### 2.1. 이름 입력
| 요소 | ID / 설명 |
|---|---|
| 라벨 | "이름" |
| 입력창 | `#portfolio-name-input` / `placeholder`: "예: 배당주 포트폴리오" (최대 20자) |
| 글자 수 | `#name-count` / 우측 하단 힌트 텍스트 "0/20자" (`.input-hint`) |

#### 2.2. 설명 입력
| 요소 | ID / 설명 |
|---|---|
| 라벨 | "설명 (선택)" |
| 텍스트 영역 | `#portfolio-desc-input` / `placeholder`: "포트폴리오에 대한 설명" (최대 100자) |
| 글자 수 | `#desc-count` / 우측 하단 힌트 텍스트 "0/100자" (`.input-hint`) |

### 3. 고정 푸터 버튼 (`.bottom-sheet-footer`)
- **취소**: 모달 닫기 (`.secondary`, `#portfolio-modal-cancel-btn`)
- **만들기**: 포트폴리오 생성 (`.primary`, `#portfolio-modal-create-btn`)

## 상호작용

| 액션 | 동작 |
|---|---|
| 핸들 드래그 다운 | 모달 닫기 |
| 이름/설명 입력 | 실시간 글자 수 갱신 |
| 만들기 버튼 | 생성 API 호출 → 성공 시 목록 갱신 |
| 취소/닫기/백드롭 탭 | 모달 닫기 |

## 상태

| 상태 | UI |
|---|---|
| 로딩 중 | 만들기 버튼 스피너 처리 |
| 이름 미입력 | 만들기 버튼 비활성화 (disabled) |
| 5개 초과 | 에러 메시지 표시 |
| 에러 | 화면 상단 또는 폼 하단 에러 텍스트/토스트 표시 |
