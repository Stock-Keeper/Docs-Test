# 요약 카드 (Summary Card)

## 개요

주요 지표를 그리드 형태로 표시하는 요약 컴포넌트.
포트폴리오 목록과 상세 화면에서 자산 현황 표시에 사용.

## 레이아웃

```
┌─────────────────────────────────┐
│  총 평가금액     총 매입금       │
│  ₩10,000,000   ₩9,000,000      │
│                                 │
│  수익금          수익률          │
│  ₩1,000,000    +11.1%          │
│                                 │
│  현금                           │
│  ₩1,000,000 · $500             │
└─────────────────────────────────┘
```

## 변형 (Variants)

### 1. 상세 요약 (Full)

- 용도: portfolio/detail
- 표시: 총 평가금액, 총 매입금, 수익금, 수익률, 현금

### 2. 목록 요약 (Compact)

- 용도: portfolio/list
- 표시: 종목 수, 총 평가금액, 수정일

## 컴포넌트 요소

| 요소 | 설명 | 표시 위치 |
|------|------|----------|
| 총 평가금액 | 종목 + 현금 합계 | Detail |
| 총 매입금 | 종목 매입금 합계 + 현금 | Detail |
| 수익금 | 평가금액 - 매입금 | Detail |
| 수익률 | (수익금 / 매입금) × 100 | Detail |
| 현금 | 원화 + 달러 | Detail |
| 종목 수 | n개 종목 | List |
| 수정일 | 상대 시간 | List |

## 스타일 속성

| 요소 | 값 |
|------|-----|
| border-radius | 16px |
| padding | 20px |
| background | var(--card-bg) |
| grid | 2열 (모바일), 4열 (태블릿) |

## 색상 규칙

| 조건 | 색상 |
|------|------|
| 수익 (양수) | 빨간색 (var(--success)) |
| 손실 (음수) | 파란색 (var(--error)) |

## 사용 화면

- `portfolio/list`: 포트폴리오 카드 내 간략 정보
- `portfolio/detail`: 상단 자산 요약 영역
