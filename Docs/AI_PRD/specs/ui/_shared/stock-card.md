# 종목 카드 (Stock Card)

## 개요

보유 종목 정보를 표시하는 카드 컴포넌트.
포트폴리오 상세, 검색 결과, 리밸런싱 등 다양한 화면에서 사용.

## 레이아웃

```text
┌────────────────────────────────────────┐
│ 삼성전자 005930          ₩11,357,500  ×│
│                         +5.2%          │
│                                        │
│ 현재 35%                    목표 30%   │
│ [───████████████┼──────────]           │
│ +17% 초과                              │
└────────────────────────────────────────┘
```

## 변형 (Variants)

### 1. 포트폴리오 종목 카드 (Full)

- 용도: portfolio/detail
- 표시: 종목명, 종목코드, 평가금액, 수익률, 비중(현재/목표), 괴리율 바, 괴리율 태그, 삭제 버튼
- 액션: 탭(삭제 버튼 클릭 시 삭제)

### 2. 검색 결과 카드 (Compact)

- 용도: stock/search
- 표시: 종목명, 코드, 현재가, 등락률, 시장
- 액션: 탭(추가 모달)

### 3. 리밸런싱 카드 (Action)

- 용도: rebalancing/check
- 표시: 종목명, 현재/목표 비중, 필요 조정량
- 액션: 없음 (정보만 표시)

## 컴포넌트 요소

| 요소 | 설명 | 조건 |
|---|---|---|
| 종목명 | 종목 이름 | 필수 |
| 종목코드 | 티커/코드 | 포트폴리오/검색 카드 |
| 삭제 버튼 | 종목 리스트에서 제거 (`×`) | 포트폴리오 카드만 |
| 평가금액 | 현재 총 평가액 | 포트폴리오 카드만 |
| 수익률 / 등락률 | 평가금액 증감률 또는 등락률 | 둘 다 |
| 목표 비율 | 목표 N% | 포트폴리오 카드만 |
| 현재 비율 | 현재 N% | 포트폴리오 카드만 |
| 괴리율 바 | 현재 vs 목표 시각화 막대 | 포트폴리오 카드만 |
| 괴리율 태그 | 초과, 미달, 적정 범위 텍스트 | 포트폴리오 카드만 |
| 등락률 | 전일 대비 변동 | 검색 카드만 |
| 시장 | KOSPI/KOSDAQ/NYSE 등 | 검색 카드만 |
| 보유 배지 | 이미 보유 시 표시 | 검색 카드만 |

## 스타일 속성

| 요소 | 값 |
|---|---|
| border-radius | 12px |
| padding | 16px |
| background | `var(--card-bg)` |

## 색상 규칙

| 조건 | 색상 |
|---|---|
| 상승 (양수) | 빨간색 (`var(--success)`) |
| 하락 (음수) | 파란색 (`var(--error)`) |
| 보합 (0) | 회색 (`var(--muted)`) |

## 사용 화면

- `portfolio/detail`: 포트폴리오 종목 목록
- `stock/search`: 검색 결과 목록
- `rebalancing/check`: 조정 필요 종목 목록
