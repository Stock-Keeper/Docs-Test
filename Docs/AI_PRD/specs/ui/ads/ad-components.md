# 광고 컴포넌트 (Ad Components)

## 개요

앱 전반에 노출되는 광고 컴포넌트의 레이아웃 및 동작 명세 (P1: 하단 배너, P2: 전면 광고, P3: 네이티브/알림 배너)

## 레이아웃

### 전면 광고 (Splash Interstitial) — P2

```
┌─────────────────────────────────┐
│                                 │
│        [스플래시 로고]            │
│                                 │
│  ┌─────────────────────────────┐│
│  │                             ││
│  │     전면 이미지 광고          ││
│  │     (Interstitial)          ││
│  │                             ││
│  │              [X 닫기 / 스킵] ││
│  └─────────────────────────────┘│
│                                 │
└─────────────────────────────────┘
```

### 하단 배너 (P1) + 네이티브 인피드 (P3) + 알림 배너 (P3)

```
┌─────────────────────────────────┐
│  메인 콘텐츠 영역                │
│  ┌─ 커뮤니티 피드 ─────────────┐│
│  │ 게시물 #1                   ││
│  │ 게시물 #2                   ││
│  │ 게시물 #3                   ││
│  │ 게시물 #4                   ││
│  │ 게시물 #5                   ││
│  │ ┌─ 네이티브 광고 ──────────┐ ││
│  │ │ Ad · 광고                │ ││
│  │ │ [광고 콘텐츠]            │ ││
│  │ └─────────────────────────┘ ││
│  │ 게시물 #6                   ││
│  └─────────────────────────────┘│
├─────────────────────────────────┤
│  [         하단 배너 광고       ] │
└─────────────────────────────────┘

┌─────────────────────────────────┐
│  ← 알림                         │
├─────────────────────────────────┤
│  [    알림 상단 고정 배너 광고   ] │
├─────────────────────────────────┤
│  알림 카드 #1                    │
│  알림 카드 #2                    │
│  ...                             │
└─────────────────────────────────┘
```

## 컴포넌트

### 1. Splash Interstitial (전면 광고) — P2

| 속성     | 값                                        |
| -------- | ----------------------------------------- |
| 위치     | 앱 콜드 스타트 시, 메인 화면 진입 전      |
| 타임아웃 | 3초 (로드 실패 시 즉시 메인 진입)         |
| 빈도     | `config.frequency_cap` 준수 (기본 2회/일) |
| 닫기     | X 버튼 또는 자동 스킵                     |

### 2. Bottom Banner (하단 배너) — P1

| 속성 | 값                                    |
| ---- | ------------------------------------- |
| 위치 | 메인 홈, 관심 종목, 자산 현황 탭 하단 |
| 크기 | 320×50 또는 Smart Banner              |
| 갱신 | `config.refresh_interval_sec` (30초)  |
| 지속 | 탭 이동 간에도 유지                   |

### 3. Feed Native Ad (네이티브 인피드) — P3

| 속성   | 값                                           |
| ------ | -------------------------------------------- |
| 위치   | 커뮤니티 피드 게시물 사이                    |
| 삽입   | `config.feed_interval` (5) 번째 아이템마다   |
| 디자인 | 게시물 카드와 동일 레이아웃 (Custom Layout)  |
| 라벨   | **[필수]** 'Ad' 또는 '광고' 명시 (정책 준수) |

### 4. Notification Sticky Banner (알림 배너) — P3

| 속성 | 값                      |
| ---- | ----------------------- |
| 위치 | 알림 센터 페이지 상단   |
| 고정 | 스크롤 시에도 상단 고정 |
| 크기 | 가로 전체 × 높이 고정   |

## 상호작용

| 액션               | 동작                               |
| ------------------ | ---------------------------------- |
| 전면 광고 닫기     | 메인 화면 진입                     |
| 전면 광고 타임아웃 | 즉시 메인 화면 진입 (3초)          |
| 배너 광고 탭       | 광고주 랜딩 페이지 (외부 브라우저) |
| 네이티브 광고 탭   | 광고주 랜딩 페이지 (외부 브라우저) |
| 알림 배너 탭       | 광고주 랜딩 페이지 (외부 브라우저) |

## 상태

| 상태              | UI                              |
| ----------------- | ------------------------------- |
| 광고 로딩 중      | 빈 공간 유지 (깜빡임 방지)      |
| 광고 로드 실패    | 해당 영역 숨김 (빈 공간 제거)   |
| 광고 비활성화     | 해당 영역 렌더링 하지 않음      |
| 네트워크 오프라인 | 캐시된 광고 표시 또는 영역 숨김 |
