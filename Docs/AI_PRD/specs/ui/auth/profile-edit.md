# 프로필 편집 화면 (Profile Edit)

## 개요
 
설정에서 진입하는 프로필 정보 편집 화면

```
┌─────────────────────────────────┐
│  [←]      프로필 편집               │
│                                 │
│                                 │
│         ┌─────────┐             │
│         │  📷    │             │
│         └─────────┘             │
│        [사진 변경하기]           │
│                                 │
│                                 │
│  닉네임                          │
│  ┌─────────────────────────┐    │
│  │ 투자왕                   │    │
```

## 컴포넌트

### 1. 헤더 (`screen-header`)
| 속성 | 값 |
|------|-----|
| 뒤로가기 | 설정 화면으로 복귀 (클릭 시 `profile-edit-cancel-modal` 다이얼로그 호출) |
| 타이틀 | "프로필 편집", 18px, Semi-bold(600), Text Primary |

### 2. 프로필 이미지 (`profile-image-section`)

| 속성 | 값 |
|---|---|
| 크기 | 100x100 px |
| 형태 | 원형 (`border-radius: 50%`), `2px` 테두리(`border-color`) |
| 기본값 | 임시 아이콘 📷 (`36px`, `opacity: 0.5`) |
| 편집 | P1에서는 변경 불가 (안내 문구만 표시, "사진 변경하기") |

### 3. 닉네임 입력 (`profile-form > form-group`)
| 속성 | 값 |
|---|---|
| 라벨 | "닉네임" |
| 초기값 | 현재 닉네임 |
| 폼 스타일 | 내부 여백 `14px 16px`, 테두리 반경 `var(--radius-md)` |
| 최대 길이 | 20자 |
| 글자수 표시 | "n/20" (우측 정렬, Text Tertiary, 12px) |

### 4. 투자 성향 (`toggle-btn-group`)

| 속성 | 값 |
|---|---|
| 라벨 | "투자 성향" |
| 옵션 | 안정형 / 중립형 / 공격형 |
| 형태 | 가로 배열 Flex, 간격 `10px` |
| 선택 상태 | Gradient Primary 배경, 흰색 텍스트 |
| 미선택 상태 | 표면색 배경 (`bg-surface`), 호버 시 Primary 옅은 배경/보더 전환 |
| 초기값 | 현재 설정값 |

### 5. 저장하기 버튼 (`profile-save-section`)

| 속성 | 값 |
|---|---|
| 위치 | 스크롤 영역 최하단 마진 적용 |
| 스타일 | Primary 버튼 |
| 비활성 조건 | 변경사항 없음 (`disabled` 속성. 닉네임이나 투자성향 변경 시 활성화) |

## 상호작용

| 액션 | 동작 |
|------|------|
| 닉네임 수정 | 글자수 업데이트, 저장 버튼 활성화 |
| 성향 변경 | 저장 버튼 활성화 |
| 저장하기 클릭 | API 호출 → 성공 시 이전 화면 복귀 |
| 뒤로가기 | 변경사항 있으면 "저장하지 않고 나가시겠습니까?" 다이얼로그 |

## 상태

| 상태 | UI |
|---|---|
| 기본 | 버튼 비활성화 |
| 변경됨 | 버튼 활성화 |
| 로딩 중 | 버튼에 스피너, 입력 비활성화 |
| 성공 | 토스트 "저장되었습니다" → 이전 화면 |
| 에러 | 토스트 메시지 |
