# 프로필 입력 화면 (Profile Input)

## 개요
최초 로그인 후 프로필 입력

## 레이아웃

```
┌─────────────────────────────────┐
│  [←]      프로필 설정               │
│                                 │
│                                 │
│         ┌─────────┐             │
│         │  📷    │             │
│         └─────────┘             │
│        [사진 변경하기]           │
│                                 │
│                                 │
│  닉네임 *                        │
│  ┌─────────────────────────┐    │
│  │ 닉네임을 입력해주세요       │    │
│  └─────────────────────────┘    │
│                         0/20    │
│                                 │
│  투자 성향 *                     │
│  ┌────────┐┌────────┐┌────────┐ │
│  │ 안정형 ││[중립형]││ 공격형 │ │
│  └────────┘└────────┘└────────┘ │
│                                 │
│                                 │
│    ┌─────────────────────────┐  │
│    │        시작하기          │  │
│    └─────────────────────────┘  │
│    나중에 변경할 수 있습니다      │
│                                 │
└─────────────────────────────────┘
```

## 컴포넌트

### 1. 헤더 (`screen-header`)

- 뒤로가기 버튼 (`[←]`): `40x40px`, 투명 배경, 클릭 시 로그아웃 확인 다이얼로그 호출
- 타이틀: "프로필 설정", 18px, Semi-bold(600), Text Primary

### 2. 프로필 이미지 (`profile-image-section`)

| 속성 | 값 |
|---|---|
| 크기 | 100x100 px |
| 형태 | 원형 (`border-radius: 50%`), `2px` 테두리(`border-color`) |
| 기본값 | 임시 아이콘 📷 (`36px`, `opacity: 0.5`) |
| 동작 | 호버 시 Primary 컬러 테두리로 강조, 클릭 축소 효과 (`scale(0.95)`) |
| 힌트 텍스트 | "사진 변경하기", 14px, Text Tertiary (`cursor: not-allowed`) |

### 3. 닉네임 입력 (`profile-form > form-group`)

| 속성 | 값 |
|---|---|
| 라벨 | "닉네임 *" (별표 기호는 Success 컬러로 표시) |
| 입력 폼 | 내부 여백 `14px 16px`, 테두리 반경 `var(--radius-md)` |
| 플레이스홀더 | "닉네임을 입력해주세요" (Text Tertiary) |
| 최대 길이 | 20자 |
| 글자수 표시 | "0/20" (우측 정렬, Text Tertiary, 12px) |

### 4. 투자 성향 (`toggle-btn-group`)

| 속성 | 값 |
|---|---|
| 라벨 | "투자 성향 *" |
| 옵션 | 안정형 / 중립형 / 공격형 |
| 형태 | 가로 배열 Flex, 간격 `10px` |
| 선택 상태 | Gradient Primary 배경, 흰색 텍스트, 그림자 없음 |
| 미선택 상태 | 표면색 배경 (`bg-surface`), 보더 유지, 호버 시 Primary 옅은 배경/보더 전환 |

### 5. 시작하기 버튼 및 안내 영역 (`profile-submit-section`)

| 속성 | 값 |
|---|---|
| 배치 | 스크롤 영역 최하단 마진 적용 |
| 버튼 속성 | 화면 하단 Primary 버튼 |
| 버튼 비활성화 | 닉네임 미입력 시 `opacity: 0.5`, 오버레이/포인터 효과 제거 |
| 힌트 문구 | "나중에 변경할 수 있습니다" (12px, Text Tertiary, 중앙 정렬) |

## 상호작용

| 액션 | 동작 |
|------|------|
| 닉네임 입력 | 실시간 글자수 업데이트 |
| 시작하기 클릭 | API 호출 → 홈 화면 이동 |
| 뒤로가기 | "로그아웃하시겠습니까?" 다이얼로그 |

## 유효성 검증

| 조건 | 메시지 |
|------|--------|
| 닉네임 비어있음 | 버튼 비활성화 |
| 20자 초과 | 입력 제한 (추가 입력 불가) |

## 상태

| 상태 | UI |
|------|-----|
| 기본 | 버튼 비활성화 |
| 닉네임 입력됨 | 버튼 활성화 |
| 로딩 중 | 버튼에 스피너, 입력 비활성화 |
| 성공 | 홈 화면 이동 |
| 에러 | 토스트 메시지 |
