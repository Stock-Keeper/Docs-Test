# 프로필 입력 화면 (Profile Input)

## 개요
최초 로그인 후 프로필 입력

## 레이아웃

```
┌─────────────────────────────────┐
│  ←                              │
│                                 │
│       프로필 설정                │
│                                 │
│         ┌─────────┐             │
│         │  📷    │             │
│         │ 프로필  │             │
│         └─────────┘             │
│        [사진 변경하기]           │
│                                 │
│  닉네임 *                        │
│  ┌─────────────────────────┐    │
│  │                         │    │
│  └─────────────────────────┘    │
│  0/20                           │
│                                 │
│                                 │
│                                 │
│    ┌─────────────────────────┐  │
│    │      시작하기            │  │
│    └─────────────────────────┘  │
│                                 │
│    나중에 변경할 수 있습니다      │
│                                 │
└─────────────────────────────────┘
```

## 컴포넌트

### 1. 헤더
- 뒤로가기 버튼 (←): 로그아웃 확인 다이얼로그
- 타이틀: "프로필 설정", 18px, Bold

### 2. 프로필 이미지
| 속성 | 값 |
|------|-----|
| 크기 | 100x100 px |
| 형태 | 원형 |
| 기본값 | Google 프로필 사진 |
| 편집 | P1에서는 변경 불가 (안내 문구만 표시) |

### 3. 닉네임 입력
| 속성 | 값 |
|------|-----|
| 라벨 | "닉네임 *" |
| 플레이스홀더 | "닉네임을 입력해주세요" |
| 최대 길이 | 20자 |
| 글자수 표시 | "0/20" (우측 하단) |

### 4. 시작하기 버튼
| 속성 | 값 |
|------|-----|
| 너비 | 화면 90% |
| 높이 | 48px |
| 스타일 | Primary 버튼 |
| 비활성 조건 | 닉네임 미입력 |

### 5. 안내 문구
- "나중에 변경할 수 있습니다"
- 12px, Secondary

## 상호작용

| 액션 | 동작 |
|------|------|
| 닉네임 입력 | 실시간 글자수 업데이트 |
| 시작하기 클릭 | API 호출 → 홈 화면 이동 |
| 뒤로가기 | "로그아웃하시겠습니까?" 다이얼로그 |

## 유효성 검증

| 조건 | 메시지 |
|------|--------|
| 닉네임 비어있음 | 버튼 비활성화 |
| 20자 초과 | 입력 제한 (추가 입력 불가) |

## 상태

| 상태 | UI |
|------|-----|
| 기본 | 버튼 비활성화 |
| 닉네임 입력됨 | 버튼 활성화 |
| 로딩 중 | 버튼에 스피너, 입력 비활성화 |
| 성공 | 홈 화면 이동 |
| 에러 | 토스트 메시지 |
