# 온보딩 화면 (Onboarding)

## 개요

앱 최초 실행 시 1회 노출되는 소개 화면, 4개 슬라이드로 앱 핵심 가치 전달

## 진입 조건

| 조건 | 동작 |
|------|------|
| 최초 실행 (`isFirstVisit=false`) | 온보딩 표시 |
| 재실행 (`isFirstVisit=true`) | 스킵 → 로그인 |
| 앱 삭제 후 재설치 | 온보딩 다시 표시 |

## 레이아웃

```
┌─────────────────────────────────┐
│                                 │
│           [아이콘]              │
│                                 │
│         [슬라이드 제목]          │
│                                 │
│         [슬라이드 설명]          │
│                                 │
│          ● ○ ○ ○               │
│      (페이지 인디케이터)         │
│                                 │
│  [← 이전]          [건너뛰기 →] │
└─────────────────────────────────┘
```

## 슬라이드 구성

### Slide 1: 핵심 가치

| 요소 | 값 |
|------|-----|
| 아이콘 | 📊 |
| 제목 | 왜 리밸런싱이 중요할까요? |
| 설명 | 목표 비율을 유지하면 리스크를 줄이고 수익을 안정화할 수 있어요 |

### Slide 2: 포트폴리오 생성

| 요소 | 값 |
|------|-----|
| 아이콘 | 📁 |
| 제목 | 나만의 포트폴리오를 만들어요 |
| 설명 | + 버튼을 눌러 새 포트폴리오를 쉽게 추가할 수 있어요 |

### Slide 3: 종목 추가 및 목표 비중

| 요소 | 값 |
|------|-----|
| 아이콘 | 🎯 |
| 제목 | 종목을 추가하고 목표 비중을 설정 |
| 설명 | 원하는 비율만 입력하면 리밸런싱은 자동으로 계산해줘요 |

### Slide 4: 시작하기

| 요소 | 값 |
|------|-----|
| 아이콘 | 🚀 |
| 제목 | 스마트한 투자 관리, 시작! |
| 설명 | 임계값 초과 시 알림을 받고 언제든 리밸런싱하세요 |

## 컴포넌트

### 1. 슬라이드 컨테이너

| 속성 | 값 |
|------|-----|
| 형태 | Carousel (스와이프 가능) |
| 전환 효과 | Fade (0.4s) |

### 2. 아이콘

| 속성 | 값 |
|------|-----|
| 크기 | 80px |
| 애니메이션 | float (3s, 상하 10px) |

### 3. 페이지 인디케이터

| 속성 | 값 |
|------|-----|
| 형태 | 도트 4개 |
| 활성 상태 | Primary 색상, scale(1.2) |
| 상호작용 | 클릭 시 해당 슬라이드 이동 |

### 4. 네비게이션 버튼

| 버튼 | 동작 | 표시 조건 |
|------|------|----------|
| ← 이전 | 이전 슬라이드 | Slide 2 이상 |
| 건너뛰기 → | 로그인 화면 이동 | Slide 1~3 |
| 시작하기 | 로그인 화면 이동 | Slide 4 |

## 상호작용

| 액션 | 동작 |
|------|------|
| 좌/우 스와이프 | 슬라이드 이동 |
| 인디케이터 탭 | 해당 슬라이드로 이동 |
| 이전 버튼 탭 | 이전 슬라이드 |
| 건너뛰기 탭 | 로그인 화면으로 즉시 이동 |
| 시작하기 탭 (Slide 4) | 로그인 화면으로 이동 |

## 로컬 스토리지

| Key | Type | 설명 |
|-----|------|------|
| isFirstVisit | boolean | 완료/건너뛰기 시 `true`로 설정 |

## 엣지 케이스

| 상황 | 처리 |
|------|------|
| 중간에 앱 종료 | 다음 실행 시 처음부터 |
| 네트워크 오류 | 영향 없음 (로컬 UI) |
