# 온보딩 화면 (Onboarding)

## 개요

앱 최초 실행 시 1회 노출되는 소개 화면, 4개 슬라이드로 앱 핵심 가치 전달

## 진입 조건

| 조건 | 동작 |
|---|---|
| 최초 실행 (`isFirstVisit=false`) | 온보딩 표시 |
| 재실행 (`isFirstVisit=true`) | 스킵 → 로그인 |
| 앱 삭제 후 재설치 | 온보딩 다시 표시 |

## 레이아웃

```text
┌─────────────────────────────────┐
│                                 │
│           [아이콘]              │
│                                 │
│         [슬라이드 제목]          │
│                                 │
│         [슬라이드 설명]          │
│                                 │
│          ● ○ ○ ○               │
│      (페이지 인디케이터)         │
│                                 │
│  [← 이전]          [건너뛰기 →] │
└─────────────────────────────────┘
```

## 슬라이드 구성

### Slide 1: 핵심 가치

| 요소 | 값 |
|---|---|
| 아이콘 | 📊 |
| 제목 | 왜 리밸런싱이 중요할까요? |
| 설명 | 목표 비율을 유지하면 리스크를 줄이고 수익을 안정화할 수 있어요 |

### Slide 2: 포트폴리오 생성

| 요소 | 값 |
|---|---|
| 아이콘 | 📁 |
| 제목 | 나만의 포트폴리오를 만들어요 |
| 설명 | + 버튼을 눌러 새 포트폴리오를 쉽게 추가할 수 있어요 |

### Slide 3: 종목 추가 및 목표 비중

| 요소 | 값 |
|---|---|
| 아이콘 | 🎯 |
| 제목 | 종목을 추가하고 목표 비중을 설정 |
| 설명 | 원하는 비율만 입력하면 리밸런싱은 자동으로 계산해줘요 |

### Slide 4: 시작하기

| 요소 | 값 |
|---|---|
| 아이콘 | 🚀 |
| 제목 | 스마트한 투자 관리, 시작! |
| 설명 | 임계값 초과 시 알림을 받고 언제든 리밸런싱하세요 |

## 컴포넌트

### 1. 슬라이드 컨테이너

| 속성 | 값 |
|---|---|
| 형태 | Carousel (`overflow: hidden`, 내부 컨테이너 `display: flex`) |
| 전환 효과 | Opacity 및 Transform 기반 Fade 효과 (`opacity 0.4s ease, transform 0.4s ease`) |

### 2. 아이콘

| 속성 | 값 |
|---|---|
| 크기 | 80px |
| 마진 | 하단 32px |
| 애니메이션 | float (3s, ease-in-out, 상하 10px 반복) |

### 3. 페이지 인디케이터

| 속성 | 값 |
|---|---|
| 형태 | 도트 4개 (크기 `10x10px`, 간격 `10px`, 배경 `Text Disabled`) |
| 활성 상태 | Primary 색상, 확대 보정 (`scale(1.2)`) |
| 전환 효과 | 배경색 및 변형 애니메이션 부여 (`transition: all 0.3s`) |

### 4. 네비게이션 버튼 (이전 / 건너뛰기)

| 속성 | 스타일 |
|---|---|
| 형태 | 투명 배경 (`background: transparent`), 패딩 `12px 16px` |
| 폰트 | 15px, Medium(500), Text Secondary |
| 동작 | 호버 시 Text Primary 로 색상 변경 |

### 5. 시작하기 버튼 (마지막 슬라이드 전용)

| 속성 | 스타일 |
|---|---|
| 테마 | Gradient Primary 배경, 흰색 텍스트 |
| 형태 | 테두리 반경 `var(--radius-lg)`, 패딩 `14px 32px` |
| 폰트 | 16px, Semi-bold(600) |
| 효과 | 기본 그림자 (`box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1)`), 호버 시 상단 이동 및 Primary 색상 그림자 강조 |

## 상호작용

| 액션 | 동작 |
|---|---|
| 좌/우 스와이프 | 슬라이드 이동 |
| 인디케이터 탭 | 해당 슬라이드로 이동 |
| 이전 버튼 탭 | 이전 슬라이드 |
| 건너뛰기 탭 | 로그인 화면으로 즉시 이동 |
| 시작하기 탭 (Slide 4) | 로그인 화면으로 이동 |

## 로컬 스토리지

| Key | Type | 설명 |
|---|---|---|
| isFirstVisit | boolean | 완료/건너뛰기 시 `true`로 설정 |

## 엣지 케이스

| 상황 | 처리 |
|---|---|
| 중간에 앱 종료 | 다음 실행 시 처음부터 |
| 네트워크 오류 | 영향 없음 (로컬 UI) |
