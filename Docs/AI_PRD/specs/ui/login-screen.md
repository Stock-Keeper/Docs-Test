# 로그인 화면 (Login Screen)

## 개요
Google OAuth 로그인 화면

## 화면 구성

```
┌─────────────────────────────────┐
│                                 │
│         [앱 로고 이미지]          │
│                                 │
│       Stock Keeper              │
│    포트폴리오 리밸런싱 앱         │
│                                 │
│                                 │
│    ┌─────────────────────┐      │
│    │  [G] Google로 시작하기  │    │
│    └─────────────────────────┘   │
│                                 │
│                                 │
│    로그인하면 서비스 이용약관 및    │
│    개인정보처리방침에 동의하게      │
│    됩니다.                       │
│                                 │
└─────────────────────────────────┘
```

## 컴포넌트

### 1. 앱 로고
- 크기: 120x120 px
- 위치: 상단 중앙

### 2. 앱 타이틀
- 메인: "Stock Keeper" (24px, Bold)
- 서브: "포트폴리오 리밸런싱 앱" (14px, Secondary)

### 3. Google 로그인 버튼
- 스타일: Google 브랜드 가이드라인 준수
- 크기: 가로 80%, 높이 48px
- 텍스트: "Google로 시작하기"
- 아이콘: Google 로고 (왼쪽)

### 4. 약관 동의 문구
- 크기: 12px, Text Secondary
- 링크: "서비스 이용약관", "개인정보처리방침"

## 상호작용

| 액션 | 동작 |
|------|------|
| 버튼 클릭 | Google OAuth 플로우 시작 |
| 로그인 성공 (신규) | 프로필 입력 화면 이동 |
| 로그인 성공 (기존) | 홈 화면 이동 |
| 로그인 실패 | 에러 토스트 표시 |

## 에러 처리

| 상황 | 메시지 |
|------|--------|
| 네트워크 오류 | "인터넷 연결을 확인해주세요" |
| OAuth 취소 | (무시, 화면 유지) |
| OAuth 실패 | "로그인에 실패했습니다. 다시 시도해주세요" |
