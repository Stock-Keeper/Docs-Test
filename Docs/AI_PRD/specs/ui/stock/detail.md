# 종목 상세 화면 (Stock Detail)

## 개요

개별 종목의 상세 정보와 보유/비율 현황을 확인하고 수정/삭제할 수 있는 상세 페이지

## 컴포넌트

### 1. 헤더

| 요소 | 설명 | Phase |
|------|------|-------|
| 뒤로가기 (←) | 포트폴리오 상세로 복귀 | P1 |
| 종목명 + 코드 | 화면 타이틀 | P1 |
| 수정 아이콘 (✏️) | 종목 정보 수정 모달 | P1 |

### 2. 현재가 섹션

| 요소 | 설명 | Phase |
|------|------|-------|
| 현재가 | 실시간 가격 (더미/캐시) | P1 |
| 등락률 | 전일 대비 퍼센트 | P1 |
| 기준 시각 | 마지막 업데이트 시각 | P1 |
| 색상 | 상승: 빨강, 하락: 파랑, 보합: 회색 | P1 |

### 3. 차트 영역 [P2]

> [!NOTE]
> 차트 영역은 P2에서 구현. P1에서는 플레이스홀더 표시.

| 요소 | 설명 | Phase |
|------|------|-------|
| 플레이스홀더 | "주가 차트 / 추후 지원 예정" | P1 |
| 캔들 차트 | OHLC 봉차트 (시가/고가/저가/종가) | P2 |
| 기간 탭 | 1일, 1주, 1달, 3달 | P2 |

### 4. 내 보유 현황 카드

| 요소 | 설명 | Phase |
|------|------|-------|
| 보유 수량 | 현재 보유 주식 수 (N주) | P1 |
| 평균 매입가 | 매수 평균가 | P1 |
| 평가금액 | 현재가 × 보유수량 | P1 |
| 수익금 | 평가금액 - 매입총액 | P1 |
| 수익률 | (현재가 - 평균매입가) / 평균매입가 × 100 | P1 |

### 5. 비율 현황 카드

| 요소 | 설명 | Phase |
|------|------|-------|
| 현재 비율 | 포트폴리오 내 현재 비중 + 진행바 | P1 |
| 목표 비율 | 설정된 목표 비중 + 진행바 | P1 |
| 괴리율 경고 | 임계값 초과 시 경고 표시 | P1 |

### 6. 액션 버튼

| 요소 | 설명 | Phase |
|------|------|-------|
| 수량 수정 버튼 | 수량/비율 수정 모달 표시 | P1 |
| 삭제 버튼 | 종목 삭제 확인 다이얼로그 | P1 |

## 상호작용

| 액션 | 동작 | Phase |
|------|------|-------|
| 뒤로가기 | 포트폴리오 상세 화면으로 복귀 | P1 |
| 수량 수정 | 수정 모달 표시 (수량, 평균매입가, 목표비율) | P1 |
| 삭제 | 확인 다이얼로그 → 삭제 처리 | P1 |
| 기간 탭 클릭 | history API 호출 → 차트 갱신 | P2 |

## 상태

| 상태 | UI | Phase |
|------|-----|-------|
| 로딩 중 | 전체 스켈레톤 | P1 |
| 삭제 확인 | 확인/취소 다이얼로그 | P1 |
| 삭제 처리 중 | 버튼 로딩 상태 | P1 |
| 차트 로딩 중 | 차트 영역 스켈레톤 | P2 |
| 차트 실패 | "차트를 불러올 수 없습니다" | P2 |

## 데이터 소스

| 항목 | 소스 | Phase |
|------|------|-------|
| 종목 기본 정보 | 포트폴리오 상세에서 전달받은 state | P1 |
| 현재가 | price API (캐시 30초) or 더미 | P1 |
| 차트 데이터 | history API | P2 |
