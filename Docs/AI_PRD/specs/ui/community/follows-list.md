# 팔로워 및 팔로잉 목록 화면 (Follows List Screen)

## 개요
내 프로필이나 타인의 프로필에서 '팔로워' / '팔로잉' 숫자를 터치했을 때 나타나는 상세 유저 리스트업 화면.

## 레이아웃

```
┌─────────────────────────────────────┐
│  ← 투자왕김철수                      │
├─────────────────────────────────────┤
│  [ 팔로워 156 ] [ 팔로잉 23 ]       │
├─────────────────────────────────────┤
│  🔍 닉네임 검색                       │
├─────────────────────────────────────┤
│                                     │
│  [프사] 배당러버          [팔로잉]   │
│  [프사] 테슬라주주        [팔로우]   │
│  [프사] 단타왕가즈아      [팔로잉]   │
│  [프사] 10년장투만함      [팔로우]   │
│                                     │
│  [ 더 불러오기 ... ]                │
└─────────────────────────────────────┘
```

## 컴포넌트 동작
- **상단 탭**: 팔로워(나를 팔로우한 사람)와 팔로잉(내가 팔로우한 사람) 목록 전환.
- **액션 버튼**: 리스트 우측에 맞팔로우(팔로우/팔로잉) 상태 버튼 토글 제공.
  - 내 프로필의 목록을 보는 중일 때 바로 [팔로우 취소] ↔ [팔로우] 가 가능하도록 구성.
- **프로필 진입**: 리스트의 닉네임이나 프사 영역 탭 시 해당 유저의 타인 프로필 상세보기로 라우팅.
