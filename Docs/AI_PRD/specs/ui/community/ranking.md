# 커뮤니티 랭킹 화면 (Rankings Screen)

## 개요
커뮤니티 내 활동 점수(좋아요, 포트폴리오 복사 등)를 기반으로 사용자 순위를 보여주는 전용 랭킹 페이지.

## 레이아웃

```
┌─────────────────────────────────────┐
│  ← 커뮤니티 랭킹                    │
├─────────────────────────────────────┤
│  [전체] [종합] [좋아요] [복사]      │
│  [주간] [월간] [누적]               │
├─────────────────────────────────────┤
│                                     │
│      👑 1위 워렌버핏동생            │
│  2위 단타의신     3위 가상화폐왕     │
│  (TOP 3 하이라이트 UI)              │
│                                     │
├─────────────────────────────────────┤
│  내 랭킹: 42위 (종합 1,234점)       │
├─────────────────────────────────────┤
│                                     │
│  4. 배당러버            1,100점     │
│  5. 10년존버            1,050점     │
│  6. 삼전가즈아            900점     │
│  ...                                │
│                                     │
│  [ 더 불러오기 ... ]                │
└─────────────────────────────────────┘
```

## 컴포넌트 동작
- **필터 1 (카테고리)**: 종합(OVERALL), 좋아요(LIKE), 복사(COPY)
- **필터 2 (기간)**: 주간(WEEKLY), 월간(MONTHLY), 누적(ALL_TIME)
- **Top 3 랭킹**: 상단에 트로피/금은동 아이콘을 사용해 크게 강조 노출
- **랭커 진입 (사용자 프로필)**: 순위에 있는 사용자를 탭하면 `profile.md` 상세 보기 화면으로 이동.

## 연관 규격
- API: `/api/community/rankings` 호출 및 필터 파라미터 연동.
