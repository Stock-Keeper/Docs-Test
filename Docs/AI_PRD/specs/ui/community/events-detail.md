# 이벤트 상세 및 참가 화면 (Event Detail Screen)

## 개요
특정 이벤트를 자세히 보고 "참가하기" 버튼을 누르거나, 대회 현황 리더보드를 확인합니다.

## 컴포넌트 동작

### 1. 참가 버튼 영역 (하단 고정바)
- **미참여 (진행중)**: `[ 참가 신청하기 ]`
  - 탭 시 수익률 대회의 경우 **내 포트폴리오를 고르는 Bottom Sheet 모달** 노출
  - 포폴 선택 시 `/api/community/events/{id}/join` 호출
- **미참여 (마감/종료됨 )**: `[ 참여 마감 ]` (비활성)
- **참여 완료**: `[ 이미 참가 신청을 완료했습니다 ]` (비활성)

### 2. 리더보드 영역
- 대회 타입(PORTFOLIO_CONTEST 등)일 경우만 노출.
- 내 순위 하이라이트.
- `전체보기` 탭 시 API를 통해 랭킹 전체를 페이지 단위로 보여주는 전체화면 팝업/모달 전개.

## 연관 규격
- `/api/community/events/{eventId}/leaderboard` API 응답 적용 (내 순위, 리더보드)
