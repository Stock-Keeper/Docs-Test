# 검색 화면 (Search Screen)

## 개요
커뮤니티 통합 검색 (종목, 게시글, 사용자)

## 레이아웃

```text
┌──────────────────────────────────────┐
│  ← [🔍 검색어 입력...]          [✕] │  ← 헤더 (`search-header`)
├──────────────────────────────────────┤
│                                      │
│  종목                       더보기 →│  ← 섹션 영역 (`result-section`)
│  ┌──────────────────────────────┐│
│  │ 삼성전자              70,000 ││  ← 결과 항목 (`result-item`)
│  │ 005930                +2.5%  ││
│  └──────────────────────────────┘│
│                                      │
│  게시글                     더보기 →│
│  ┌──────────────────────────────┐│
│  │ 삼성전자 실적 분석           ││  ← 게시글 항목 (`post-result-item`)
│  │ ❤️ 42                        ││
│  └──────────────────────────────┘│
│                                      │
│  사용자                     더보기 →│
│  ┌──────────────────────────────┐│
│  │ 🧑 삼성러버                  ││  ← 사용자 요소 (`user-row`)
│  │ 팔로워 123                   ││
│  └──────────────────────────────┘│
│                                      │
└──────────────────────────────────────┘
```

## 컴포넌트

### 1. 검색 헤더 (`.search-header`)
- 뒤로가기 (`#community-search-back-btn`)
- 검색 입력필드 (`#community-search-input`, 자동 포커스 됨)
- 검색어 지우기 (`#search-clear-btn`)

### 2. 종목 섹션 (`#result-stocks` / `.result-section`)
| 요소 | 설명 | Phase |
|---|---|---|
| 종목명 (`.result-name`) | 종목명 표시 | P1 |
| 종목코드 (`.result-code`) | 종목코드 표시 | P1 |
| 현재가 (`.price`) | 현재 주가 | P1 |
| 등락률 (`.change`) | `up` (상승) / `down` (하락) 스타일 분기 | P1 |
| 더보기 (`.more-link`) | 카테고리별 전체 결과 화면 진입 | P1 |

### 3. 게시글 섹션 (`#result-posts` / `.result-section`)
| 요소 | 설명 | Phase |
|---|---|---|
| 제목 (`.post-title`) | 말줄임 등 적용된 게시글 제목 | P1 |
| 좋아요 (`.post-likes`) | ❤️ N | P1 |
| 더보기 (`.more-link`) | 게시글 전체 결과 화면 진입 | P1 |

### 4. 사용자 섹션 (`#result-users` / `.result-section`)
| 요소 | 설명 | Phase |
|---|---|---|
| 프로필 (`.user-avatar-sm`) | 캐릭터 아이콘 (작은 사이즈) | P1 |
| 닉네임 (`.result-name`) | 사용자명 | P1 |
| 추가정보 | 하위 정보로 팔로워 등 노출 (상세 구조 TBD) | P1 |
| 더보기 (`.more-link`) | 사용자 전체 결과 화면 진입 | P1 |

## 상호작용

| 액션 | 동작 |
|------|------|
| 검색어 입력 | 500ms 디바운스 후 검색 |
| 종목 클릭 | 해당 종목 포함 포트폴리오 리스트 |
| 게시글 클릭 | 게시글 상세 화면 |
| 사용자 클릭 | 프로필 화면 |
| 더 보기 클릭 | 해당 카테고리 전체 결과 (페이지네이션) |

## 상태

## 상태 (`#search-content` 내부)

| 상태 | ID/컨테이너 | UI |
|---|---|---|
| 초기 | `#search-initial` | 텍스트: "종목, 게시글, 사용자를 검색해보세요" |
| 검색 중 | `#search-loading` | 스피너 (`⏳`), "검색 중..." |
| 결과 항목 | `#search-results` | 종목/게시글/사용자 카테고리별 노출 |
| 결과 없음 | `#search-empty` | 아이콘 (`🔍`), "검색 결과가 없습니다" |
| 에러 | - | 에러 메시지 + 재시도 (API 연동 시) |
