# 공유 바텀 시트 (Share Bottom Sheet)

## 개요

포트폴리오나 게시글 상세 화면에서 `[공유]` 버튼을 누르면 밑에서 올라오는 공통 바텀 시트. 
외부 SNS 또는 메신저로 콘텐츠를 전파할 수 있게 함.

## 레이아웃

```
┌─────────────────────────────────┐
│  (반투명 배경 오버레이)           │
│                                 │
├─────────────────────────────────┤
│  ────                           │
│                                 │
│  친구에게 공유하기                │
│  ─────────────────────────────  │
│                                 │
│  [ 🔗 링크 복사 ]                │
│  [ 🟡 카카오톡으로 공유 ]         │
│  [ 📱 기타 (Native Share) ]     │
│                                 │
│             [취소]               │
└─────────────────────────────────┘
```

## 컴포넌트 동작

| 요소 | 설명 | 플랫폼 네이티브 대응 |
|------|------|-----------------------|
| 링크 복사 | `/share` API를 호출해 얻은 짧은 URL을 클립보드 복사 유틸에 복사함. "링크가 복사되었습니다" 토스트 노출 | 클립보드 API 연동 |
| 카카오톡 공유 | 카카오톡 SDK의 커스텀 템플릿 호출로 ogImage, ogTitle, 접속 버튼 전달 | React Native Kakao SDK 연동 |
| 기타 공유 | OS 디바이스 기본 공유 창 UI (iOS 활동 컨트롤러, 안드로이드 인텐트 쉐어) 띄움 | React Native Share 연동 |

## 상호작용
| 액션 | 동작 |
|------|------|
| 각 공유 수단 탭 | 1) 스켈레톤 상태 하에 공유 API 호출. 2) 응답 수신 후 해당 외부 앱 진입 또는 클립보드 복사. |
| 취소 혹은 배경 탭 | 바텀 시트 내려가며 닫힘. |
