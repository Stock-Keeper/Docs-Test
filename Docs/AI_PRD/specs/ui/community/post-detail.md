## 컴포넌트

### 1. 헤더 (`.post-detail-header`)

| 요소 | ID / 설명 | Phase |
|---|---|---|
| 뒤로가기 | `#post-detail-back-btn` / 이전 화면 복귀 (`←`) | P1 |
| 카테고리 | `#post-detail-category` / 게시글 말머리 (예: 국내주식) | P1 |
| 더보기 | `#post-detail-more-btn` / 게시글 옵션(수정/삭제 등) 바텀시트 노출 (`⋮`) | P1 |

### 2. 프로필 영역 (`.post-author-section`)

| 요소 | ID / 설명 | Phase |
|---|---|---|
| 프사 | `#post-author-avatar` / 클릭 → 프로필 상세로 이동 | P1 |
| 닉네임 | `#post-author-name` / 작성자 닉네임 | P1 |
| 날짜 및 메타 | `#post-date`, `#post-follower-count` / n분 전, 팔로워 수 표시 | P1 |
| 수정됨 배지 | `#post-edited` / 수정된 글이면 표시 (`.edited-badge`) | P1 |
| 팔로우 버튼 | `#post-follow-btn` / 팔로우/팔로잉 상태 토글 (`.following` 클래스 추가) | P1 |

### 3. 게시글 본문 영역 (`.post-content-area`)

| 요소 | ID/클래스 / 설명 | Phase |
|---|---|---|
| 포트폴리오 종목 | `#portfolio-stocks-section` (`.portfolio-card`) / 카테고리인 경우 노출, 종목 비율 슬라이드 | P1 |
| 제목 | `#post-title` / 본문 제목 | P1 |
| 내용 | `#post-text` / 본문 내용 | P1 |
| 이미지 | `#post-images` / 첨부된 이미지 레이아웃 형태 표시 | P1 |
| 종목 태그 | `#stock-tags` / 연동된 종목 태그 목록 표기 (`.stock-tag`) | P1 |
| 통계 및 액션 | `.post-actions` / 좋아요 (`#post-like-count`), 댓글 (`#post-comment-count`), 재게시 (`#post-repost-count`), 공유 (`#post-share-btn`) 버튼 묶음 | P1 |
| 다른 글 캐러셀 | `#author-posts-carousel` / 사용자의 이전 글 제공 | P1 |

### 4. 댓글 영역 (`.comments-section`)

| 요소 | 클래스/ID / 설명 | Phase |
|---|---|---|
| 정렬 드롭다운 | `#comment-sort` (`.comment-sort-select`) / 최신순(기본), 인기순, 날짜순 | P1 |
| 댓글 아이템 | `.comment-item` / 프사, 닉네임, 시간, 내용 묶음 | P1 |
| 댓글 좋아요 | `.comment-like-count` 및 액션 버튼(`❤️`) | P1 |
| 대댓글 작성 | `.reply-btn` (`↩️`) / 스레드 혹은 본문 멘션 댓글 | P1 |

### 5. 댓글 작성 (고정 영역 `.comment-input-area`)

- 하단에 고정됨
- 입력란 (`#comment-input`) 및 전송 버튼 (`➤`, `#comment-submit-btn`)
- ↩️ 클릭 시 `@닉네임` 자동 입력 (일반 댓글로 등록, 스레드 X)

## 상호작용

| 액션 | 동작 |
|------|------|
| 프사 탭 | 프로필 화면 이동 |
| 팔로우 탭 | 팔로우/언팔로우 토글 |
| 좋아요 탭 | 좋아요 토글 |
| 공유 탭 | 공유 시트 표시 |
