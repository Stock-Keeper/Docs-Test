# 게시글 상세 화면 (Post Detail Screen)

## 개요

게시글 본문, 댓글/대댓글 표시

## 레이아웃

```
┌─────────────────────────────────────┐
│  ←          전체/국내/해외          │  ← 헤더 (고정)
├─────────────────────────────────────┤
│ [프사] 투자왕김철수                 │  ← 프로필 영역 (고정)
│       1월 15일 (수정됨) · 팔로워 3K │
│                          [팔로우]   │
├─────────────────────────────────────┤
│                                     │  ← 스크롤 영역
│  삼성전자, 지금 사도 될까요?        │  ← 제목
│                                     │
│  안녕하세요. 최근 삼성전자 주가가   │  ← 본문
│  많이 올랐는데요...                 │
│                                     │
├─────────────────────────────────────┤
```

## 컴포넌트

### 1. 프로필 영역

| 요소        | 설명                        |
| ----------- | --------------------------- |
| 프사        | 클릭 → 프로필 상세          |
| 닉네임      | 작성자 닉네임               |
| 날짜        | n분 전 / m월 d일 (7일 이후) |
| (수정됨)    | 수정된 글이면 표시          |
| 팔로워 수   | N / N.NK / N.NM             |
| 팔로우 버튼 | 팔로우/팔로잉 토글          |

### 2. 댓글 영역

| 요소          | 설명                           |
| ------------- | ------------------------------ |
| 정렬 드롭다운 | 최신순(기본) / 인기순 / 날짜순 |
| 댓글 구성     | 프사 / 닉네임 · 시간 / 내용    |
| ❤️            | 댓글 좋아요                    |
| ↩️            | 대댓글 작성                    |

### 3. 대댓글 작성

- ↩️ 클릭 시 `@닉네임` 자동 입력
- 일반 댓글로 등록 (스레드 X)

## 상호작용

| 액션      | 동작                 |
| --------- | -------------------- |
| 프사 탭   | 프로필 화면 이동     |
| 팔로우 탭 | 팔로우/언팔로우 토글 |
| 좋아요 탭 | 좋아요 토글          |
| 공유 탭   | 공유 시트 표시       |

### P3 추가) 포트폴리오 첨부 컨테이너

게시글에 포트폴리오가 첨부된 경우 본문 하단에 표시됩니다.

```
┌─────────────────────────────────────┐
│  [ 📂 첨부된 포트폴리오 요약 ]      │
│  - 전체 수익률: +15.2%              │
│  - 종목 수: 5개                     │
│                                     │
│  [ 카피하기 (🪙 3 소모) ]          │
├─────────────────────────────────────┤
│  [ 🔒 상세 분석 지표 (프리미엄) ]   │
│                                     │
│  AI 평가 점수: [ 블러 처리됨 ]      │
│  변동성/Sharpe: [ 블러 처리됨 ]     │
│                                     │
│  [ 상세 지표 열람 (🪙 2 소모) ]    │
└─────────────────────────────────────┘
```

| 액션                | 동작                                                                                                     |
| ------------------- | -------------------------------------------------------------------------------------------------------- |
| 카피하기 클릭       | 토큰/멤버십 확인 모달 → 복사 성공 모달 → 포트폴리오 탭 이동 권유 (희망 비중만 복사, 실제 매수 연동 없음) |
| 상세 지표 열람 클릭 | 별도 unlock API 호출 → 토큰 2개 차감 동의 모달 → 블러 해제                                               |
