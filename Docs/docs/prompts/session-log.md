# 세션 논의 기록

## 언제

사용자가 요청할 때:

- "논의 내용 정리해줘"
- "세션 로그 작성해줘"

---

## 형식

```markdown
# 세션 로그: [주제]

## 날짜
YYYY-MM-DD

## 배경
[어떤 작업 중 논의 시작]

## 논점

### 1. [질문]
- 문제:
- 논의:
- 결론:

## 결정사항
- 결정 1
- 결정 2

## 팀원 공유 요약
> [핵심 2-3줄]

## 후속 작업
- [ ] 할 일
```

---

## 저장 위치

`docs/decisions/SESSION-YYYYMMDD-주제.md`
