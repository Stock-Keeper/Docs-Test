# 📱 Stock-Keeper 와이어프레임 가이드 (Figma 초보용)

> 도형만 그릴 줄 알아도 충분해요! 단계별로 따라하세요.

---

## 🎯 시작 전 세팅

### 1. 새 파일 만들기

1. Figma 접속 → **New design file**
2. 빈 캔버스에서 시작

### 2. 모바일 프레임 만들기

1. 좌측 상단 **Frame 도구 (F키)**
2. 우측 패널에서 **Phone > iPhone 14 (390×844)** 선택
3. 7개 화면 = 7개 프레임 복사해서 가로로 배치

```
[로그인] [프로필] [홈] [상세] [검색] [리밸런싱] [설정]
```

---

## 🎨 사용할 색상 (3개만!)

| 용도 | 색상 | Hex |
|------|------|-----|
| 배경 | 진한 회색 | `#1a1a2e` |
| 카드/박스 | 연한 회색 | `#2d2d44` |
| 텍스트 | 흰색 | `#FFFFFF` |

> 💡 색상 입력: 도형 선택 → 우측 **Fill** → 색상 클릭 → Hex 입력

## 📐 기본 수치 (외우세요!)

| 항목 | 크기 |
|------|------|
| 화면 가로 여백 | 좌우 20px씩 |
| 요소 간 간격 | 12px 또는 16px |

## 🔧 화면별 그리는 법

### 화면 1: 로그인

```
┌─────────────────────┐
│      (상태바)        │  ← 44px 높이, 생략해도 됨
│                     │
│       [로고]        │  ← 100×100 사각형, 둥글기 28
│     Stock-Keeper    │  ← 텍스트 36px
│    스마트한 포트폴리오  │  ← 텍스트 15px, 회색
│                     │
│  💼  📈  ⚖️         │  ← 아이콘 3개 가로 배치
│                     │
│ ┌─────────────────┐ │
│ │ Google로 시작하기  │ │  ← 흰색 버튼, 높이 48px
│ └─────────────────┘ │
│                     │
│   이용약관 | 개인정보  │  ← 텍스트 11px
└─────────────────────┘
```

**만드는 순서:**

1. 배경색 `#1a1a2e` 프레임
2. 사각형(100×100, 둥글기 28) → 로고
3. 텍스트 "Stock-Keeper" (36px, Bold)
4. 흰색 사각형(가로 full - 40px, 높이 48px) → 버튼

## 💡 꿀팁

1. **헤더 먼저** 만들고 모든 화면에 복사
2. **카드 하나** 완벽하게 만들고 복제
3. 모르겠으면 **HTML 프로토타입 켜놓고** 보면서 그리기
4. 완벽하지 않아도 됨 - **레이아웃만 전달**되면 OK!
