## 🎯 전체 디자인 원칙

### 1. 정보 계층 (Information Hierarchy)

```
Level 1: 즉시 인지 (1초 이내) - 큰 숫자, 색상, 아이콘
Level 2: 스캔 가능 (3초 이내) - 카드, 리스트, 요약
Level 3: 탐색 시 확인 - 상세 정보, 설정값
```

### 2. 투자 앱 색상 심리학 (한국 시장 기준)

| 상태 | 색상 | HEX 예시 | 이유 |
|-----|------|---------|-----|
| 상승/수익 | 🔴 빨강 | `#E53935` | 한국 증시 관례 |
| 하락/손실 | 🔵 파랑 | `#1E88E5` | 한국 증시 관례 |
| 중립/변동없음 | ⚫ 회색 | `#9E9E9E` | 변화 없음 표시 |
| 경고/주의 | 🟡 노랑 | `#FFC107` | 리밸런싱 필요 |

### 3. 터치 타겟

- **최소 터치 영역**: 44×44px (iOS), 48×48dp (Android)
- **버튼 간격**: 최소 8px
- **리스트 아이템 높이**: 최소 56px (정보량에 따라 72~88px)

## 🔐 Page 1: 로그인

### UX 패턴 권장

| 요소 | 권장 방식 | 이유 |
|-----|----------|-----|
| 로고 | 앱 아이콘 + 텍스트 로고 분리 | 브랜드 인지도 + 가독성 |
| 기능 소개 | 아이콘 + 1줄 텍스트 (3개) | 인지 부하 최소화 |
| CTA 버튼 | 화면 하단 고정, 전체 너비 | 엄지 접근성 (Thumb Zone) |

```
1️⃣ 앱 로고 (상단 1/3)
2️⃣ 가치 제안 텍스트 (중앙)
3️⃣ 로그인 버튼 (하단)
```

### 🎨 디자인 팁

- **배경**: 단순 그라데이션 or 미묘한 패턴 (종목 그래프 실루엣 등)
- **로딩 상태**: 버튼 내 스피너 (페이지 전환 아님)
- **에러**: 토스트 메시지 (모달 X)

## 👤 Page 2: 프로필 설정

### UX 패턴 권장

| 요소 | 권장 방식 | 이유 |
|-----|----------|-----|
| 프로필 사진 | 선택사항으로 축소 or 제거 | MVP에서 불필요한 마찰 제거 |
| 닉네임 입력 | 실시간 유효성 검사 + 글자 수 표시 | 즉각적 피드백 |
| 투자 성향 | Chip 선택 (탭으로 전환) | 한 번에 한 개만 선택 가능 표현 |
| 진행 버튼 | 유효성 통과 전 비활성화 | 에러 방지 |

### 입력 필드 상태

```
Default  → 회색 테두리
Focus    → Primary 컬러 테두리 + 라벨 상단 이동
Valid    → 초록 체크 아이콘 (우측)
Error    → 빨강 테두리 + 에러 메시지 하단 표시
```

### 🎨 디자인 팁

- **건너뛰기**: 우측 상단 텍스트 버튼 (회색)
- **투자 성향 설명**: 각 칩 탭 시 짧은 툴팁 or 하단 설명 변경

## 📊 Page 4: 포트폴리오 상세 (★ 핵심)

### 🔑 비율 표현: 진행도 vs 초과/미달

이 앱에서 가장 중요한 UI 결정 중 하나입니다.

#### 방식 A: 진행도 바 (Progress Bar)

```
삼성전자  ████████████████░░░░  35%  (목표 30%)
          [=====현재=====][목표 마커]
```

- **장점**: 직관적, 전체 대비 현재 위치 파악 용이
- **단점**: "얼마나 초과/미달인지" 한눈에 안 보임
- **적합한 경우**: 전체 비율 분포를 보고 싶을 때

#### 방식 B: 초과/미달 바 (Deviation Bar) ✅ 권장

```
삼성전자      ━━━|████▌     +5%p 초과
              목표↑
                   
카카오        ▌████|━━━     -3%p 미달
                  목표↑
```

- **장점**: 리밸런싱 필요성이 즉시 보임, 방향성 명확
- **단점**: 전체 비율은 별도 확인 필요
- **적합한 경우**: 리밸런싱 앱 (우리 앱의 핵심 목적!)

#### 방식 C: 숫자 하이라이트 (가장 심플)

```
삼성전자   현재 35%  →  목표 30%  [+5%p]
                              ↑ 빨간색 강조
```

- **장점**: 가장 명확, 복잡한 시각화 불필요
- **단점**: 시각적 재미 없음

### 📐 최종 권장: 하이브리드 방식

```
┌─────────────────────────────────────────┐
│ 삼성전자                          ₩11.3M │
│ 005930                           +5.2%  │
│                                         │
│ 현재 35%  ━━━━|██▌        목표 30%       │
│               ↑ 중심점                   │
│           [+5%p 초과]  ← 빨간색 태그     │
└─────────────────────────────────────────┘
```

**핵심 요소**:

1. **중심 기준 바**: 목표 비율이 중심선, 좌우로 미달/초과 표시
2. **명시적 숫자 태그**: "+5%p 초과" 또는 "-3%p 미달"
3. **색상 코딩**: 초과(매도필요)=파랑, 미달(매수필요)=빨강

### 종목 카드 레이아웃 옵션

#### 옵션 A: 테이블 형식 (현재)

```
종목명          비율              평가금액
삼성전자     [=====|==]35%     ₩11.3M +5.2%
SK하이닉스   [====|]25%        ₩8.1M +8.1%
```

- **장점**: 정보 밀도 높음, 비교 용이
- **단점**: 좁은 화면에서 복잡해 보임

#### 옵션 B: 카드 리스트 (권장)

```
┌─────────────────────────────────────┐
│ 삼성전자 005930              ₩11.3M │
│ ━━━━|██ [+5%p 초과]           +5.2% │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│ SK하이닉스 000660             ₩8.1M │
│ ━━━|  [적정 범위 ✓]           +8.1% │
└─────────────────────────────────────┘
```

- **장점**: 터치 영역 명확, 개별 종목 구분 쉬움
- **단점**: 스크롤 필요

**👉 권장**: 종목 5개 이하는 **테이블**, 6개 이상은 **카드 리스트**

## ⚖️ Page 6: 리밸런싱 분석 (★ 핵심)

### 차트 선택: 도넛 vs 카드

#### 현재: 도넛 차트 2개 (나란히)

```
  [현재]           [목표]
   ╭───╮          ╭───╮
  ╱ ▓▓▓ ╲        ╱ ▓▓ ╲
 │ ▓   ▒ │      │ ▓  ▒ │
  ╲ ░░░ ╱        ╲ ░░░ ╱
   ╰───╯          ╰───╯
```

- **문제**: 직접 비교가 어려움, 종목별 차이 파악 힘듦
- **적합한 경우**: 전체적인 분포 변화를 보고 싶을 때

#### 권장: 가로 비교 바 차트 ✅

```
삼성전자   현재 ████████████▓▓  35%
           목표 ████████████     30%
                              [-5%p]

SK하이닉스 현재 ██████████      25%
           목표 ██████████      25%
                              [적정]

카카오     현재 ████▓           12%
           목표 ██████          15%
                              [+3%p]
```

- **장점**: 종목별 현재 vs 목표 직접 비교 가능
- **단점**: 종목 많으면 스크롤 필요
