### 사용자가 보고 싶어하는 정보

| 정보 | 중요도 | 표현 수준 권장 | 이유 |
|-----|--------|---------------|-----|
| 꼭 입력해야 하는 항목 | ⭐⭐⭐ 높음 | 필수/선택 구분 명확히 | 불필요한 입력 요구 시 이탈률 증가 |
| 입력 조건/제한 | ⭐⭐ 중간 | 인풋 하단 힌트 텍스트 | 에러 방지, UX 향상 |
| 건너뛰기 옵션 | ⭐⭐ 중간 | 필요시 표시 | 급한 사용자 배려 |

### 기획 시 고려사항

- **최소한의 입력만 요구**: 닉네임만 필수, 나머지는 옵션
- **나중에 수정 가능함을 안내**: 완벽하지 않아도 진행할 수 있다는 심리적 안정감
- **투자 성향의 활용**: 현재는 수집만 하고 활용처 불분명 → Phase 1에서 활용 계획 필요

### 🤔 추가 논의 필요

2. 프로필 사진은 MVP에 필요한가? (Google 프로필 자동 연동?)
3. 온보딩 튜토리얼을 여기에 넣을 것인가?
