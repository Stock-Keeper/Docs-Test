## 📋 화면 개요

| 항목               | 내용                                    |
| ------------------ | --------------------------------------- |
| **화면명**         | 앱 소개 온보딩                         |
| **목적**           | 신규 사용자에게 앱의 핵심 가치 전달    |
| **진입 경로**      | 앱 최초 실행 (비회원 상태)             |
| **이동 가능 화면** | → 로그인/회원가입                      |

### 사용자 스토리

> "신규 사용자로서 나는 앱이 무엇을 할 수 있는지 **3초 안에 파악**하고, 가입할지 결정하고 싶다."

## 💡 설계 가이드라인

### 1. 슬라이드 구성 (예시)

#### 슬라이드 1: 환영

```
┌─────────────────────────────────────┐
│                                     │
│              📊                     │
│                                     │
│      Stock-Keeper에 오신 것을       │
│           환영합니다!               │
│                                     │
│   스마트한 포트폴리오 관리의 시작   │
│                                     │
│           ● ○ ○                     │
│                                     │
│                        [건너뛰기 →] │
└─────────────────────────────────────┘
```

#### 슬라이드 2: 핵심 기능 1

```
┌─────────────────────────────────────┐
│                                     │
│           [일러스트]                │
│                                     │
│     목표 비율 설정으로              │
│     체계적인 포트폴리오 관리        │
│                                     │
│           ○ ● ○                     │
│                                     │
│     [← 이전]          [건너뛰기 →] │
└─────────────────────────────────────┘
```

#### 슬라이드 3: 핵심 기능 2

```
┌─────────────────────────────────────┐
│                                     │
│           [일러스트]                │
│                                     │
│     실시간 리밸런싱 제안으로        │
│        언제나 균형 잡힌 투자        │
│                                     │
│           ○ ○ ●                     │
│                                     │
│     [← 이전]           [시작하기]  │
└─────────────────────────────────────┘
```

### 2. 디자인 고려사항

| 항목 | 권장 |
| ---- | ---- |
| 슬라이드 수 | 2~3개 (너무 많으면 이탈) |
| 애니메이션 | 페이드 또는 슬라이드 전환 |
| 일러스트 | 앱 테마 색상과 일관성 유지 |
| 하단 인디케이터 | 도트 형태 (현재 위치 표시) |

### 3. 건너뛰기 동작

- 모든 슬라이드에서 "건너뛰기" 버튼 노출
- 건너뛰기 시 즉시 로그인 화면으로 이동
- 온보딩 완료 여부 로컬 저장 (다시 보지 않기)
